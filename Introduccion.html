1. Introducción general

La Ciencia de Datos es un área que combina programación, análisis estadístico y conocimiento del problema para transformar datos en información útil. Su objetivo es encontrar patrones, explicar comportamientos y apoyar la toma de decisiones.

Dentro de este campo, el Machine Learning permite construir modelos que aprenden a partir de ejemplos, ya sea con respuestas conocidas o descubriendo estructuras ocultas en los datos.

¿Para qué sirve?

Para analizar datos, detectar patrones, hacer predicciones y automatizar decisiones basadas en evidencia.

¿Qué se necesita?

Datos de buena calidad, herramientas de programación, comprensión del problema y métodos de análisis adecuados.

¿Qué se aprende?

A limpiar datos, explorarlos, modelarlos, evaluarlos y comunicar resultados de forma clara.

3. Aprendizaje supervisado

En el aprendizaje supervisado, el modelo entrena con datos que ya tienen una respuesta conocida. Es decir, cada ejemplo incluye variables de entrada y una salida esperada.

Entrada (X) → salida conocida (y)
Clasificación

Se utiliza cuando la variable objetivo es una categoría.

Ejemplos: spam / no spam, fraude / no fraude, ataque / normal.

Regresión

Se utiliza cuando la variable objetivo es un valor numérico continuo.

Ejemplos: precio, temperatura, ventas, demanda estimada.

Diferencia principal

Clasificación
Predice clases o categorías.
Regresión
Predice valores numéricos continuos.

4. Aprendizaje no supervisado

En el aprendizaje no supervisado, los datos no tienen una variable objetivo. El modelo busca descubrir relaciones, grupos o estructuras escondidas dentro del conjunto de datos.

Clustering

Agrupa observaciones similares. Sirve para segmentar clientes, documentos, usuarios o comportamientos.

Reducción de dimensionalidad

Reduce la cantidad de variables sin perder demasiada información. Muy útil para visualizar y simplificar datos.

Reglas de asociación

Busca relaciones frecuentes entre variables. Se usa mucho en análisis de canasta de compras.

5. ¿Qué es Jupyter Lab?

Jupyter Lab es un entorno interactivo donde se puede escribir código, ejecutar análisis, documentar el proceso y mostrar resultados dentro de un mismo archivo llamado notebook.

Es muy utilizado en clases y proyectos de Ciencia de Datos porque mezcla código, texto explicativo, tablas, gráficos y conclusiones en un formato ordenado y reproducible.

Un notebook suele combinar celdas de código y celdas Markdown.
# Ejemplo simple en Jupyter Lab import pandas as pd df = pd.read_csv("datos.csv") df.head()

Ventajas de Jupyter Lab

Interactividad

Permite ejecutar bloques de código por partes y ver resultados al instante.

Documentación

Integra explicaciones con Markdown, ideal para clases y reportes.

Visualización

Facilita mostrar tablas, gráficos y salidas de modelos en el mismo entorno.

6. Python en Ciencia de Datos

Python es uno de los lenguajes más utilizados en Ciencia de Datos por su sintaxis simple, su gran comunidad y la enorme cantidad de librerías disponibles para análisis, visualización y Machine Learning.

En cursos introductorios suele ser la mejor opción porque permite concentrarse en los conceptos sin que la sintaxis del lenguaje sea una barrera tan grande.

7. Librerías principales y su uso

Estas son algunas de las librerías más clásicas y útiles para trabajar en Jupyter Lab con Python.

NumPy

Uso principal: cálculo numérico.

Permite trabajar con arreglos, vectores, matrices y operaciones matemáticas eficientes.

import numpy as np

numeros = np.array([1, 2, 3, 4])
print(numeros.mean())
Pandas

Uso principal: manejo de datos tabulares.

Sirve para leer archivos, limpiar información, transformar columnas y analizar DataFrames.

import pandas as pd

df = pd.read_csv("datos.csv")
print(df.head())
Polars

Uso principal: procesamiento rápido de datos.

Es una alternativa moderna para trabajar con datasets grandes de forma más eficiente.

import polars as pl

df = pl.read_csv("datos.csv")
print(df.head())
Matplotlib

Uso principal: visualización base.

Permite construir gráficos de líneas, barras, histogramas, dispersión y muchas otras visualizaciones.

import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [4, 5, 6])
plt.show()
Seaborn

Uso principal: gráficos estadísticos.

Facilita crear visualizaciones más elegantes para distribuciones, correlaciones, boxplots y heatmaps.

import seaborn as sns

sns.histplot(data=df, x="edad")
Scikit-learn

Uso principal: Machine Learning clásico.

Incluye algoritmos de clasificación, regresión, clustering, evaluación y preprocesamiento.

from sklearn.linear_model import LinearRegression

modelo = LinearRegression()
modelo.fit(X, y)
SciPy

Uso principal: funciones científicas y matemáticas.

Entrega herramientas para optimización, álgebra lineal, estadística y procesamiento numérico avanzado.

from scipy import stats

resultado = stats.norm.mean()
print(resultado)
Statsmodels

Uso principal: modelamiento estadístico.

Útil para regresiones estadísticas, pruebas de hipótesis e interpretación más formal de modelos.

import statsmodels.api as sm

modelo = sm.OLS(y, X).fit()
print(modelo.summary())
TensorFlow / PyTorch

Uso principal: Deep Learning.

Se usan para construir redes neuronales más avanzadas en problemas complejos como imágenes, texto o audio.

import tensorflow as tf

modelo = tf.keras.Sequential()
print(modelo)

9. Términos generales

Dataset

Conjunto de datos con el que se trabaja.

Variable

Cada característica o columna del dataset.

Observación

Cada fila o registro de los datos.

Feature

Variable de entrada usada para entrenar un modelo.

Target

Variable objetivo que se quiere predecir.

Modelo

Representación matemática que aprende patrones desde los datos.

Entrenamiento

Proceso en que el modelo aprende usando ejemplos.

Métrica

Medida usada para evaluar qué tan bien funciona un modelo.
